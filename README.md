# Biodata Builder

A fully client-side biodata editor that can be deployed as a static site on Vercel.

## Features

- Default view-only mode.
- Edit mode protected by the static 4-digit code `1042`.
- Save progress as a `.biodata` JSON file and reopen it later.
- Export to PDF through browser print and export to Word-compatible `.doc`.
- Upload, resize, and reposition the profile photo so it fills the photo frame.
- In-app change log panel for tracking edits during a session or inside saved files.

## Deploy On Vercel

Import this project into Vercel as a static frontend app. The deployment output directory is `public`, and no server-side code or processing is required.
